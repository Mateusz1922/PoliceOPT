Python project for implementing an optimization scenario for relocating the police stations in the city of 
Chicago based on the current crime rates among the districts. Implemented in Python.

The repository contains:
- Police_Team12_OPT - jupyter notebook file containing python implementation of the project and extraction of the data
- Optimization_for_all - a report on the results and conclusions drawn from our implementation. It contains as well the
                         the explanation of our approach and ways to extract the data from the JSON files
- Chicago_density.png, new_allocation.png - python generated pictures containing ne allocation of the stations and crime density in the city of Chicago
